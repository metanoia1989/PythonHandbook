NoSQL简介

安装包

  • 进入虚拟环境py2_db，联网安装包redis
pip install redis
  • 到中文官网-客户端下载redis包的源码，使用源码安装
unzip redis-py-master.zip
cd redis-py-master
python setup.py install

调用模块

  • 引入模块
from redis import *
  • 这个模块中提供了StrictRedis对象，用于连接redis服务器，并按照不同类型提供了不同方法，进行交互操作
