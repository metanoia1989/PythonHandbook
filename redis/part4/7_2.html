配置机器2

  • 在演示中，192.168.12.84为学生的一台ubuntu机器的ip，为了演示方便，使用ssh命令连接
ssh 192.168.12.84
  • 在192.168.12.84上进入Desktop目录，创建redis目录
  • 在redis目录下创建文件7003.conf，编辑内容如下
port 7003
bind 192.168.12.84
daemonize yes
pidfile 7003.pid
cluster-enabled yes
cluster-config-file 7003_node.conf
cluster-node-timeout 15000
appendonly yes
  • 在redis目录下创建文件7004.conf，编辑内容如下
port 7004
bind 192.168.12.84
daemonize yes
pidfile 7004.pid
cluster-enabled yes
cluster-config-file 7004_node.conf
cluster-node-timeout 15000
appendonly yes
  • 在redis目录下创建文件7005.conf，编辑内容如下
port 7005
bind 192.168.12.84
daemonize yes
pidfile 7005.pid
cluster-enabled yes
cluster-config-file 7005_node.conf
cluster-node-timeout 15000
appendonly yes
  • 总结：三个文件的配置区别在port、pidfile、cluster-config-file三项
  • 使用配置文件启动redis服务
redis-server 7003.conf
redis-server 7004.conf
redis-server 7005.conf
