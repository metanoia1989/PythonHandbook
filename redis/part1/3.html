NoSQL简介

服务器端

  • 服务器端的命令为redis-server
  • 可以使用help查看帮助文档
redis-server --help
  • 推荐使用服务的方式管理redis服务
  • 启动
sudo service redis start
  • 停止
sudo service redis stop
  • 重启
sudo service redis restart

客户端

  • 客户端的命令为redis-cli
  • 可以使用help查看帮助文档
redis-cli --help
  • 运行测试命令
ping
  • 切换数据库
  • 数据库没有名称，默认有16个，通过0-15来标识
select 1
