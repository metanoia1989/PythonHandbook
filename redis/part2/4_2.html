键命令

  • 查找键，参数支持正则表达式
keys pattern
  • 例1：查看所有键
keys *
  • 例2：查看名称中包含a的键
keys '*a*'
  • 判断键是否存在，如果存在返回1，不存在返回0
exists key1
  • 例3：判断键'py1'、'py2'是否存在
exists 'py1'
exists 'py2'
  • 查看键对应的value的类型
type key
  • 例4：查看键'py1'的值类型，为redis支持的五种类型中的一种
type 'py1'
  • 删除键及对应的值
del key1 key2 ...
  • 例5：删除键'py3'、'py4'、'py5'、'py6'
del 'py3' 'py4' 'py5' 'py6'
  • 设置过期时间，以秒为单位
  • 如果没有指定过期时间则一直存在，直到使用DEL移除
expire key seconds
  • 例6：设置键'py1'的过期时间为10秒
expire 'py1' 10
  • 查看有效时间，以秒为单位
ttl key
  • 例7：查看键'py2'的有效时间
ttl 'py2'
