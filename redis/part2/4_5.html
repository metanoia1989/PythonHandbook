NoSQL简介

set

  • 无序集合
  • 元素为string类型
  • 元素具有唯一性，不重复
  • 说明：对于集合没有修改操作

增加

  • 添加元素
sadd key member1 member2 ...
  • 例1：向键'py21'的集合中添加元素'yg'、'xln'、'yzp'
sadd 'py21' 'yg' 'xln' 'yzp'

获取

  • 返回所有的元素
smembers key
  • 例2：获取键'py21'的集合中所有元素
smembers 'py21'

删除

  • 删除指定元素
srem key member
  • 例3：删除键'py21'的集合中元素'yzp'
srem 'py21' 'yzp'
